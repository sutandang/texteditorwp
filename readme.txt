== Description ==
Tonjoo Text Editor is the simple wordpress plugin that allow the user to use editor, it requires at least php 5.3. You can also integrate Tonjoo Text Editor with your newly created themes, so you can focus on theme usability and design. Moreover Tonjoo Text Editor supports custom ads so you can add your own ads on theme options page.

**Features :**

* Extensive theme options support : Text Area, Media Upload, Select, Select with Image, Checkbok and many more.
* Input sanitation and validation.
* Theme Integration API.
* Custom ads on options page
